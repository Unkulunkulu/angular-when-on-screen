# AngularJS directive whenOnScreen

Directive whenOnScreen triggers evaluation of given angular expression when the
element enters visible screen area.

## Usage

    <el ... when-on-screen="something()"></el>

See `index.html` in source repository or
http://plnkr.co/edit/t0MszlolwrgpxGxrZvig?p=preview for a complete example.

## Repository

    git@github.com:pnu/angular-when-on-screen.git

## Installation

    bower install angular-when-on-screen --save
